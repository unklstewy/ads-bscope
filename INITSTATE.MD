# GOALS
1. CREATE A WEB APPLICATION (PWA)  THAT WILL ALLOW ADS-B DATA FOR NEARBY AIRCRAFT SLEW A SEESTAR S30/S30-PRO/S50 TELESCOPE TO TARGET AIRCRAFT
2. APPLICATION MUST SUPPORT BOTH ALT/AZIMUTH AND EQUITORIAL MOUNTED SCOPES
3. APPLICATION SHOULD BE ABLE TO MAKE USE OF THE WIDE FIELD CAMERA ON THE S30 AND S3O PRO
4. IMAGING OPTIONS: CAPABLE OF STILL PHOTOS AND TRIGGERING VIDEO CAPTURE

# RULES
APPLICATION MUST FOLLOW MODERING PWA SECURITY FOCUSED DESIGN
APPLICATION MUST SUPPORT USERS AND ROLES
APPLICATION MUST UTILIZE AN OPENSOURCE DATABASE WITH STRONG SECURITY MODELS
APPLICATION SHOULD SUPPORT ONLINE ADS-B DATA SERVICES OR A LOCAL SOFTWARE DEFINED RADIO (RTL-SDR,HACKRF ONE, ETC)
APPLICATION MUST BE WRITTEN IN GO LANGUAGE. (GO LANGUAGE MUST BE INSTALLED HOMEBREW IS AVAILABLE)
APPLICATION MUST ALSO SUPPORT ALPACA ASCOM SIMULATOR FOR TESTING (NEEDS TO BE INSTALLED)
APPLICATION SUPPORRTING APPLICATIONS MUST BE CONTAINERIZED
APPLICATION SOURCE CODE MUST BE HEAVILY COMMENTED TO PROVIDE FUTURE INSIGHTS INTO EACH FUNCTION, METHOD, INTERFACE, STRUCT, ETC.
